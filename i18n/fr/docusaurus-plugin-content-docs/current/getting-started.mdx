# Bien demarrer

Cette section contient un exemple de documentation traduite. Elle permet de montrer comment l'equipe peut gerer des contenus multilingues sans changer le code de l'application.

Les auteurs peuvent creer des fichiers equivalents au contenu anglais dans ce dossier de locale. Une fois valides et fusionnes, ces fichiers sont publies automatiquement avec le meme pipeline CI/CD.
