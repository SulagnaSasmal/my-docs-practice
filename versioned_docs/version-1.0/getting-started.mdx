# Getting Started

Welcome to the documentation portal. This guide walks you through everything you need to go from zero to a fully running environment in under 10 minutes.

## Prerequisites

Before you begin, make sure you have the following:

| Requirement | Minimum Version |
|---|---|
| Node.js | 20.0+ |
| npm | 9.0+ |
| Git | 2.40+ |

## Quick Start

### 1. Clone the repository

```bash
git clone https://github.com/your-org/your-project.git
cd your-project
```

### 2. Install dependencies

```bash
npm install
```

### 3. Start the development server

```bash
npm run start
```

Your site is now live at [http://localhost:3000](http://localhost:3000).
